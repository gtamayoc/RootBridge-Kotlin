# Stack Overview

- **Language**: Kotlin
- **UI**: Jetpack Compose (Material 3)
- **Architecture**: Clean Architecture (UI, Core, Domain, Data, Overlay)
- **Coroutines**: Kotlin Coroutines + Flow
- **Dependency Injection**: Hilt (optional)
- **Logging**: Timber
- **Testing**: JUnit, MockK, AndroidX Test
- **Build System**: Gradle Kotlin DSL
- **Root Access**: NDK (C/C++) for memory scanning
- **Permissions**: Accompanist Permissions
- **Version SDK**: min 24, target 35, compile 36
